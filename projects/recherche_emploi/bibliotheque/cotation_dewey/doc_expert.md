# Classification et Indexation Dewey – Dossier complet pour une maîtrise experte

Objectif : te donner une compréhension **exhaustive, opérationnelle et experte** du système Dewey, pour être capable de :

* attribuer des cotes précises et cohérentes ;
* expliquer clairement ta pratique en entretien ;
* travailler en autonomie en bibliothèque ;
* comprendre les logiques profondes de la classification.

---

# 1. Introduction : à quoi sert la classification Dewey ?

La **Classification Décimale Dewey (CDD)** est un système de classement **thématique** des documents qui repose sur :

* une structure **décimale** ;
* 10 grandes classes → divisées en sous-classes → puis en subdivisions plus précises ;
* un principe : *un document = une cote = un emplacement unique*.

Elle sert à :

* organiser les rayons ;
* permettre aux usagers de retrouver les documents par sujet ;
* standardiser le rangement entre bibliothèques.

---

# 2. Structure générale de la Dewey

La Dewey repose sur **10 classes principales** :

* 000 Généralités
* 100 Philosophie, psychologie
* 200 Religion
* 300 Sciences sociales
* 400 Langues
* 500 Sciences pures
* 600 Techniques / sciences appliquées
* 700 Arts, loisirs
* 800 Littératures
* 900 Histoire, géographie

Chaque classe :

* contient 10 sous-classes (ex : 300 → 310, 320, 330…)
* elles-mêmes divisées en dizaines (ex : 320 → 321, 322, 323…)
* puis unités (ex : 323 → 323.1, 323.2…)

---

# 3. Comment construire une cote Dewey ?

Construire une cote Dewey = 3 étapes :

## Étape 1 : Identifier le sujet principal du document

Toujours se demander : **de quoi parle *principalement* ce document ?**

Exemples :

* Livre sur les abeilles → zoologie → 595.79
* Livre sur apprendre Python → informatique → 005.133
* Livre sur la parentalité → psychologie → 155.646

## Étape 2 : Trouver la classe Dewey correspondante

Grâce à :

* l’index Dewey ;
* des tableaux résumé (annexe ci-dessous) ;
* son expérience.

## Étape 3 : Ajouter des précisions si nécessaire

Réduction ou extension selon :

* la spécialité ;
* le public ;
* le niveau de précision requis ;
* la politique de la bibliothèque.

Exemples :

* Roman science-fiction → R SCI (on ne cote pas en Dewey pour la fiction adulte en France)
* Nutrition → 613.2
* Peinture française → 759.4

---

# 4. Indexation vs classification : bien faire la différence

La Dewey sert à **classifier**, c’est-à-dire donner **UNE cote**.
L’indexation sert à :

* décrire le document avec plusieurs mots-clés (RAMEAU, thésaurus interne).

En pratique :

* 1 document = 1 cote Dewey → rangement
* 1 document = X matières → recherche dans le catalogue

---

# 5. Procédure opérationnelle : comment attribuer une cote Dewey au quotidien

Voici comment travaille un·e bibliothécaire expérimenté·e :

1. **Lire la couverture** (titre + 4e de couverture) → première intuition.
2. **Identifier le sujet principal** → ne pas mélanger thèmes secondaires.
3. **Vérifier dans le tableau Dewey** → trouver la classe la plus précise.
4. **Regarder les notices d’autres bibliothèques** → pour harmoniser.
5. **Noter la cote complète** (ex : 944.04) et **vérifier les classes voisines** (ex : 944.03 / 944.05) pour éviter les erreurs de voisinage.
6. **Appliquer les règles locales** :

   * précision souhaitée (plus ou moins fine) ;
   * regroupement thématique souhaité.
7. **Attribuer la cote finale**.

---

# 6. Conseils d’expert pour être perçue comme très compétente

* Toujours vérifier **la cohérence** : un livre de cuisine ne peut pas être en 640 mais en 641.5.
* Éviter les cotes trop générales si une cote précise existe.
* Utiliser la règle : *si le document traite de plusieurs sujets, prendre le sujet dominant*.
* Vérifier les cotes dans les catalogues nationaux (BnF, SUDOC, réseaux locaux).
* Connaître les **trous** dans la Dewey (ex : 370 = éducation, mais 371 = structures scolaires, 372 = enseignement primaire…).

Formulation pour entretien :

> « Je maîtrise la classification Dewey, de l’identification du sujet principal à l’attribution d’une cote précise. Je sais également harmoniser les cotes avec les pratiques locales et vérifier la cohérence des classes voisines. »

---

# 7. Exemple de cotation détaillée

Document : *Comprendre l’anxiété chez l’adulte*

1. Sujet principal : anxiété → psychologie
2. Classe : 150 Psychologie
3. Sous-classe : 152 Processus mentaux ou 155.9 Troubles émotionnels
4. Recherche précise : anxiété = 152.46 ou 155.92 selon orientation
5. Choix : 152.46 (processus émotionnels → anxiété)
6. Cote finale : **152.46 ANX** (ANX = 3 premières lettres du titre si politique locale)
